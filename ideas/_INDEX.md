## Idea Statuses

| Status | Meaning |
|--------|---------|
| 🌱 Raw | Just captured — not yet evaluated |
| 🔍 Developing | Being fleshed out or researched |
| ✅ Validated | Worth pursuing — has a path forward |
| 📦 Shelved | Deliberately set aside — not dead, just not now |
| 🚀 Active | Being built or executed (link to project) |

---

## How to Capture an Idea

Add a new entry to the relevant file. Minimum viable entry:

```markdown
### Idea Title
**Status:** 🌱 Raw
**Added:** YYYY-MM-DD
**Summary:** One sentence.
```

Agents can surface ideas via the propose-then-write flow (see `AGENTS.md`) using `type: idea`.
