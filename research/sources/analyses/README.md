# Source: Analyses

Long-form technical analyses — yours, written by other LLMs for you, or pulled from external authors.

## Naming

`YYYY-MM-DD-short-slug.md` — e.g. `2026-05-30-push-flow-analysis.md`

## What goes here

- Multi-page architectural diagnoses
- Code review write-ups
- Performance investigations
- Tradeoff comparisons between approaches

These are denser than articles and usually written for a specific decision. After ingestion, the key claims get extracted into `research/topics/` pages so they're discoverable from any project.
