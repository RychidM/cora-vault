# Source: Articles

Web clips, blog posts, and other article-format sources.

## Naming

`YYYY-MM-DD-short-slug.md` — e.g. `2026-06-02-apple-apns-best-practices.md`

## Recommended workflow

1. Use **Obsidian Web Clipper** to capture an article as markdown into this folder
2. Optionally bind `Download attachments for current file` to a hotkey so images come down locally
3. Run `/vault-ingest <filename>` (or just say "ingest the article you just clipped")

The article file is immutable after this — never edit it. The synthesis lives in `research/topics/`.
